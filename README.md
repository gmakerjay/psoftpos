# ระบบจัดการขายหน้าร้าน (Store POS System)

ระบบจัดการจุดขาย (Point of Sale) ที่พัฒนาด้วย Python และ CustomTkinter ออกแบบมาให้มีความสวยงาม ทันสมัย ใช้งานง่าย และรองรับการทำงานแบบ Offline เต็มรูปแบบ

## 🌟 ฟีเจอร์หลัก (Key Features)

### 1. ระบบขายหน้าร้าน (POS)
*   **Barcode Scanning:** รองรับการยิงบาร์โค้ดต่อเนื่อง (Auto-focus) เพิ่มสินค้าเข้าตะกร้าได้ทันที
*   **Receipt Customization:** ใบเสร็จแบบ Minimal รองรับทั้งกระดาษความร้อน 58mm/80mm และกระดาษ A4/A5
*   **Silent Printing:** ระบบพิมพ์ใบเสร็จแบบเงียบ (Silent Print) พิมพ์ตรงไปยังเครื่องพิมพ์ความร้อน (Thermal) ผ่านคำสั่ง ESC/POS และพิมพ์เงียบผ่าน GDI (Windows Printer) เพื่อความเร็วสูงสุดและไม่เปิดหน้าต่าง PDF กวนใจ
*   **Price Types:** รองรับราคาหลายระดับ (ราคาปลีก, ราคาส่ง, ราคาพิเศษ)
*   **Customer Display:** หน้าจอแสดงผลสำหรับลูกค้า (Customer Display) เพื่อความโปร่งใส

### 2. การจัดการสินค้าและสต็อก
*   **Sticky Category:** ระบบจดจำหมวดหมู่ล่าสุดที่เลือก (Dropdown) ช่วยให้เพิ่มสินค้าหมวดเดียวกันได้อย่างรวดเร็ว
*   **Smart Entry:** ระบบ Select All อัตโนมัติเมื่อ Focus ช่องกรอกข้อมูล และรองรับปุ่ม Enter เพื่อเลื่อนไปยังช่องถัดไป
*   **Stock Alerts:** แจ้งเตือนสินค้าใกล้หมด (ระดับวิกฤต <= 3 ชิ้น)
*   **Data Import/Export:** สำรองข้อมูลสินค้าผ่านไฟล์ Text (Markdown) และรายงานยอดขายผ่าน Excel

### 3. ระบบรายงานและ Logs
*   **Enhanced Logging:** บันทึกทุกกิจกรรมสำคัญของโปรแกรม (Action Trace) เพื่อตรวจสอบข้อผิดพลาดย้อนหลังในโฟลเดอร์ `Logs/`
*   **Daily Summary:** ระบบปิดยอดรายวัน (End of Day) ที่จะล้างฐานข้อมูลเพื่อความรวดเร็ว แต่ยังคงสำรองข้อมูลเป็นไฟล์ Text ไว้เสมอ

## 💻 การรันใน IDE (VS Code, PyCharm, etc.)

หากต้องการนำโค้ดไปรันหรือพัฒนาต่อในเครื่องอื่น ให้ปฏิบัติตามขั้นตอนนี้:

### 1. สภาพแวดล้อมที่จำเป็น (Environment)
*   **Python:** แนะนำเวอร์ชัน 3.10 หรือ 3.11
*   **Libraries:** ติดตั้งผ่าน `pip install -r requirements.txt`
*   **Font:** ตรวจสอบว่ามีไฟล์ `FC Sara Samkan [Non-commercial] Bold.ttf` อยู่ในโฟลเดอร์หลัก เพื่อให้ใบเสร็จภาษาไทยแสดงผลถูกต้อง

### 2. การเริ่มใช้งาน (Setup)
1.  สร้าง Virtual Environment: `python -m venv .venv`
2.  Activate Virtual Environment: `.venv\Scripts\activate`
3.  ติดตั้ง Dependencies: `pip install -r requirements.txt`
4.  รันโปรแกรม: `python main.py`

## ⌨️ ปุ่มลัดแป้นพิมพ์ (Keyboard Shortcuts)

เพื่อความสะดวกรวดเร็วในการทำงานหน้าร้าน ระบบรองรับการสั่งการผ่านคีย์บอร์ดทั้งหมดโดยไม่ต้องใช้เมาส์:

### 1. หน้าขายสินค้า (POS)
*   **[F1]** : โฟกัสช่องค้นหาสินค้า/สแกนบาร์โค้ดด่วน (และล้างข้อมูลเดิมเพื่อยิงสแกนต่อเนื่องทันที)
*   **[F7]** : เปิดหน้าพิมพ์ส่วนลด / โฟกัสช่องส่วนลด
*   **[F8]** : เปิด/ปิดคำนวณภาษี VAT และโฟกัสช่องกรอก % VAT
*   **[F9]** : เปิดหน้าขายใหม่ (Multi-Session / พักบิลใหม่แยกอิสระ)
*   **[F10]** หรือ **[Enter]** (เมื่อพร้อมชำระเงิน): เปิดหน้าต่างรับเงิน/ชำระเงินด่วน
*   **[F11]** : ล้างรายการสินค้าทั้งหมดในตะกร้าปัจจุบัน (ยกเลิกบิล)
*   **[F12]** : ปิดหน้าขาย (Session) ปัจจุบัน

### 2. หน้าจัดการสินค้า (Products)
*   **[F5]** หรือ **[Ctrl+N]** : เปิด Wizard เพิ่มสินค้าใหม่ด่วนทีละขั้นตอน
*   **[Ctrl+F]** : โฟกัสช่องค้นหาสินค้า
*   **[Ctrl+R]** : โหลด/รีเฟรชข้อมูลในตารางสินค้า

## ⚙️ โหมดประสิทธิภาพสูง (Performance Mode)

สำหรับเครื่องคอมพิวเตอร์หน้าร้านรุ่นเก่าที่มีสเปกเครื่องจำกัด (เช่น CPU รุ่นเก่า หรือ RAM น้อย) คุณสามารถเปิดใช้งาน **LOW-END MODE** เพื่อลดภาระการทำงานและเพิ่มความเร็วของระบบได้ โดยการแก้ไขไฟล์ `performance_config.py` ในเครื่องของท่าน:

1.  เปิดไฟล์ `performance_config.py` ด้วยโปรแกรมแก้ไขข้อความ (Text Editor)
2.  ปรับเปลี่ยนค่าตัวแปร:
    ```python
    LOW_END_MODE = True  # เปลี่ยนจาก False เป็น True เพื่อเปิดโหมดประหยัดทรัพยากร
    ```
3.  **ผลลัพธ์ที่จะเกิดขึ้นทันที:**
    *   ลดจำนวนการเชื่อมต่อฐานข้อมูลใน Pool จาก 3 เหลือ 2 เพื่อลดภาระ RAM
    *   ลดการเก็บแคชรูปภาพสินค้าลงเพื่อประหยัดหน่วยความจำ
    *   เปิดระบบ Virtual Scrolling สำหรับรายการสินค้าที่ยาวเกิน 30 รายการขึ้นไป ป้องกันการแสดงผลกระตุก
    *   ลดความละเอียด DPI ของแผนภูมิรายงาน เพื่อให้การโหลดหน้าวิเคราะห์รวดเร็วขึ้น
    *   ตั้งดีเลย์การยิงค้นหาอัจฉริยะ (Debounce Delay) เพื่อไม่ให้ระบบ Query ฐานข้อมูลหนักเกินไปขณะพิมพ์พิมพ์ค้นหา

## 📦 การสร้างไฟล์ Executable (.exe)

หากต้องการบิ้วด์โปรแกรมเป็นไฟล์ .exe เพื่อนำไปใช้เครื่องที่ไม่มี Python:

1.  ติดตั้ง PyInstaller: `pip install pyinstaller`
2.  รันคำสั่งบิ้วด์ (ใช้ไฟล์ .spec ที่เตรียมไว้):
    ```bash
    pyinstaller build_exe.spec --clean --noconfirm
    ```
3.  ไฟล์โปรแกรมจะอยู่ที่โฟลเดอร์ `dist/โปรแกรมขายหน้าร้าน/`

## 🛠️ โครงสร้างไฟล์ในโปรเจกต์
*   `ui/`: ส่วนติดต่อผู้ใช้ทั้งหมด
*   `database/`: ระบบจัดการฐานข้อมูล SQLite (Row Factory & Connection Pool)
*   `utils/`: เครื่องมือช่วย (Printer, PDF, Logger, Barcode)
*   `data/`: ที่เก็บฐานข้อมูล (.db) และรูปภาพสินค้า
*   `Backup/`: ที่เก็บไฟล์สำรองข้อมูลจากการปิดยอด
*   `Logs/`: บันทึกการทำงานและข้อผิดพลาดรายวัน
